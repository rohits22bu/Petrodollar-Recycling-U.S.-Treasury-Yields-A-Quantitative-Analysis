# Petrodollar Recycling & U.S. Treasury Yields: A Quantitative Analysis

Time series analysis examining the relationship between Middle Eastern 
oil revenues and U.S. Treasury yields over 2000-2024.

## Methods
- ARIMA baseline model selection via AIC/BIC
- ARIMAX with Brent crude as exogenous variable
- Post sample forecast comparison vs. Holt-Winters and random walk

## Data
- DGS10: 10-Year Treasury Constant Maturity Rate (FRED)
- DCOILBRENTEU: Brent Crude Oil Spot Price (FRED)
